# Outcastsites

A fan-made wiki website for **Outcast Open Tibia 7.6**, built to help players find important quests, upgrades and access information in one place.

## Overview

**Outcastsites** is a static wiki-style website created to codument useful content for the Outcast server, including:

- Quest walkthroughs
- City and hunting area information
- Access quests
- Equipment upgrade guides
- Mushrooms, electric talismans and other utility references and other game-related resources

The goal of this project is to make commonly searched Outcast information easier to find, faster to browse, and more organized than scattered forum posts or in-game trial and error.

## Live site

**Website:**
Website: https://jaakkoku.github.io/Outcastsites/

## Features

- Wiki-style navigation
- Categorized game content
- Quest and area pages
- Upgrade and progression references
- Beginner-friendly structure
- Static hosting via GitHub pages

## How to use

1. Open the live site
2. Browse categories from the homepage
3. Select or search the area, quest, or upgrade you want help with
4. Use the wiki as a progression/reference guide while playing
